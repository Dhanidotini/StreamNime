# StreamNime

Website for streaming anime


## Features

- Image upload (Poster & Banners)
- Cross platform


## Demo

Live demo: https://streamnime.okutakun.my.id
## Run Locally

Clone the project

```bash
  git clone https://github.com/Dhanidotini/StreamNime
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  composer install
  pnpm install
```

Start the server

```bash
  composer run dev
```
